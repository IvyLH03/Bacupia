# Bacupia

将nga安科备份为.docx文件的脚本

目前处于开发阶段，用于备份明日方舟安科[真的假的，给我一百条命当深海猎人……？](https://bbs.nga.cn/read.php?tid=40452148)

目前可以生成四种文件：
1. 无格式版
  - 便于作者完结后进行导出排版工作
  - 删除所有换行、HTML标签、大部分论坛标签、论坛排版等
  - 对HTML和Unicode特殊字符进行自动替换
  - 删除楼主回复他人的楼层
    
2. 阅读版 
  - 保留基本格式，便于读者阅读
  - 保留换行，对骰点进行加粗
  - 添加只看楼主的楼层和每层楼的发布时间标签

3. 数据文件
  - 将所有回复楼层的数据保存为json文件
  - 保留论坛排版代码、点赞数等，便于排版过程中的人工参考
    
4. 处理过的数据
  - 将楼层内容进行初步整理后保存为json文件
  - 用于进行骰点统计、词频统计等数据分析
